Multiple Variant Images Test

Test Overview

This page tests the new multiple images per variant functionality.

What's Changed:
  • Templates now sync both front and back thumbnails to variants
  • Each variant can have multiple images (not just one)
  • The native theme gallery will show both images automatically
  • No gallery injection hacks needed!

How to Test

  1. Create/Edit a Dual-Sided Template:
    • Go to your template admin page
    • Create a new template or edit an existing one
    • Make sure it has both front and back designs
    • Save the template
  2. Check the Sync Results:
    • After saving, check the console logs
    • You should see "Syncing dual-sided template with both front and back thumbnails"
    • It should report syncing 2 images per variant
  3. Verify in Shopify Admin:
    • Go to the product in Shopify admin
    • Look at the variant that has the template assigned
    • You should see 2 images for that variant
  4. Check the Storefront:
    • View the product on your storefront
    • The product gallery should show both front and back images
    • You should be able to click between them

Expected Results

Feature Before After
Images per variant 1 (front only) 2 (front and back)
Gallery behavior Single image (no gallery) Native gallery with thumbnails
Gallery injection Required (but buggy) Not needed!
Theme compatibility Theme-specific hacks Works with any theme

Troubleshooting

If you don't see both images:

  • Check the browser console for errors
  • Make sure the template has both frontThumbnail and backThumbnail saved
  • Try manually re-syncing from the templates page
  • Clear your browser cache and reload
  • Check that the variant doesn't already have existing images that need to be removed

Console Commands

You can run these in the browser console on your product page:

// Check if there are multiple images in the gallery
document.querySelectorAll('media-gallery img').length

// Check variant images (if using Horizon theme)
document.querySelectorAll('slideshow-component img').length

// See all image sources
Array.from(document.querySelectorAll('.product img')).map(img => img.src)
